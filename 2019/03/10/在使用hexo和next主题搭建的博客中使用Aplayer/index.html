在使用hexo和next主题搭建的博客中使用Aplayer

发表于 2019-03-10 | 分类于 随笔 | 阅读次数：

在博客中使用Aplayer播放你的网易云歌单是一件很简单的事情，但很多人没有前端基础，用起来很难，这里介绍一种很简单的方式。

####下载
这里通过npm的方式：
npm install aplayer –save
下载完成后可以在你的hexo目录下找到node_modules\aplayer。复制里面的dist到themes\next\source下面。

####新建aplayer文件
在themes\next\source\js\src目录下新建aplayer.js文件，复制下面的内容进去：
() const ap = new APlayer({ element: document.getElementById('aplayer'), mutex: true, theme: '#97dffd', order: 'random', lrcType: 3, fixed: true, autoplay: true }); $.ajax({ url: 'https://api.i-meto.com/meting/api?server=netease&type=playlist&id=89017693', success: function (list) { ap.list.add(JSON.parse(list)); } }); ()
这里我们修改内容中的url，把里面的id替换成你的歌单id，这样就可以加载播放你的歌单了。
